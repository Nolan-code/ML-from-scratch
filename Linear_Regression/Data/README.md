# Data Folder

**Type:** CSV  

This folder contains all datasets used for the Bike Count Prediction project.

## Dataset Description

The initial dataset includes the following features:

- Wind speed
- Visibility
- Radiation
- Rainfall
- Snowfall
- Dew point temperature
- Temperature
- Humidity
- Holidays
- Functional days
- Seasons
- Hour

  From these, the features chosen for the project are:

- Temperature
- Humidity level
- Dew point temperature
- Radiation
- Rain
- Snow

## Data Cleaning

The dataset has been cleaned once in the notebook used for the library-based implementation. No further preprocessing has been applied in this repository.
