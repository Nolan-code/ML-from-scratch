# Data Folder

-2 sets of data, one for red wine and the other for white wine
**Type:** CSV  

This folder contains all datasets used for the wine quality project.

## Dataset Description

The initial dataset includes the following features:

-fixed acidity
-volatile acidity
-citric acidity
-residual sugar
-chlorides
-free sulfure dioxyde
-total sulfure dioxyde
-density
-pH
-sulfate
-alcohol

##Data cleaning

The data where already useable and all the features were kept.
